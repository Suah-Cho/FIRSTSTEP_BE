## 🔍 기능
```sh
- 게시판 등록/수정/보기/삭제
- 로그인/로그아웃
- 회원가입/회원탈퇴
- 마이페이지(대여물품 조회, 비밀번호 변경)
- 게시물 대여/반납
```


```bash
$ cd FIRSTSTEP_FE
$ npm install
$ npm run
```
